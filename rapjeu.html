Présentation :
Rap jeu, une série de jeu sur la culture rap présenté par Mehdi Maizi, le journaliste rap préféré des OG. Cette série est produite par la branche musical de RedBull (RedBinks), elle est disponible sur la plateforme YouTube.

Parce que c'est notre projet !
Notre projet consistait donc a recréer le jeu en format web
afin de permettre à tous les amateurs de rap de jouer à Rap Jeu.
Organisation :
Dans la réalisation du projet je me suis occuper du visuel, ce qui comprend
un reprise et une adaptation du style graphique de l'émission Rap Jeu
MEHDI MAIZI :
Pour la réalisation Mehdi Maizi il m'a fallut faire plusieurs expressions facials, histoire de donner plus de dynamisme.
Les éléments graphiques :
Il m'a fallut créer plusieurs éléments graphiques de type icone pour les questions et les autres
Les animations :
Après avoir reproduit le plateau de rap jeu, je l'ai également animé sur after effect pour ajouté une un
effet flottant aux blocs et à l'éclair, les animations des backgrounds ont été réalisés sur after effect
puis exporter au format JSON ce qui permet une intégration facile aux documents sans temps de chargement etc.
Responsive :
Pour une bonne utilisabilité du jeu nous avons adapté le format aux smartphone aux smartphones
et aux ordinateurs. Cela comprend notamment les backgrounds mais également certains éléments graphiques tel que les jauges de noms et les encadrements de textes.
